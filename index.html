Require

jQuery 1.8+. Making a 'vanilla' version isn't big deal, but in practice I use these controls where jQuery already used a lot anyway.

Valid for (i.e. tested in)

  • Chrome (webkit) 24+
  • Opera 12.15+
  • Firefox 20+
  • IE 10+

Known weakness point

List does not check available size, i.e. you have to check is enough place at the bottom and side to show list in open state. It is doesn't matter for my projects so I don't care yet.

How to use

SquareSelector is a runtime-creation kind of UI controls. This mean that you have to make new Javascript object to use it.

First step is prepare container for item(s) in html-file. Next, you need to call constructor explicitly to make and embed selector to the prepared place.

Usual way to do this is something like (in documentReady function or later):

Usage example

    var item_container = $('#selectors_container1');
    var selector1 = new LxSquareSelector({
        container: item_container,
        title: 'Size',
        subtitle: 'of paper',
        active: '90x90',
        list: [
            {
                value: '50x50',
                display: '5x5cm',
                line: '5x5 cm (tiny glass)'
            },
            {
                value: '70x70',
                display: '7x7cm',
                line: '7x7 cm (coffee or tea cup)'
            },
            {
                value: '90x90',
                display: '9x9cm',
                line: '9x9 cm (standard)'
            },
            {
                value: '90x50',
                display: '9x5cm',
                line: '9x5 cm (business card)'
            }
        ],
        callback: function (value) {
            alert('Selected paper: ' + value);
        }
    });

If you don't need future interaction with the control except callback function on changing, you can use simple constructions like new LxSquareSelector({...})

If you're planning works with selector's methods, you have to keep it into variable var my_selector = new LxSquareSelector({...}) and call published methods later, e.g., my_selector.disable()

Anyway, you have to set up control's properties, such as:

Property Optional Type What does it mean
container No string|object jQuery selector (jQuery object or text) of parent container for button
active No string currently selected VALUE (not display)
list No array of objects Array of dropdown list items, see next table
title Yes string Top text line
disabled Yes boolean true or false, state of button. Default value is false
subtitle Yes string Bottom text line
class_name Yes string Additional class name that will be added to the button while construction. Predefined classes:
  • green-button — button with green background
  • red-button — button with red background
  • blue-button — button with blue background
  • round-button — round button
  • round-button-animated — round button that rotate borders on hover (works only in modern browsers)
  • rounded-button — button with rounded corners
  • giant-button — button with size x2
custom_item Yes string Additional element that will be inserted inside button while construction. This is useful when you want to make color chooser, for example. You can see how it works here (click red button on the top of page, then change background to solid color).
callback Yes function Callback function that calls whan value changed
list_header Yes string Header of dropdown list
list_footer Yes string Footer of dropdown list
list_width Yes string Width of dropdown list in px. Default value is 200px
list_align Yes string Alignment of dropdown list relative to the button: left, right or center. Default value is center

And, last but not least, the list items itself:

Property Optional What does it mean
value No Value
display Yes Short display value - what users see on the button. If nothing set, then the value will be used.
line Yes Long display value - what user see in the list
separated Yes true or false, is this line will be separated from others? (default value is false)
disabled Yes true or false, is this line will be disabled? (default value is false)

If list contains only one value, this makes simple button (see example above, green and blue buttons).

Public methods

Method Parameters What does it mean
actualize(value) value, string Set current button value according to given one
disable() — Disable button
enable() — Enable button
disabled() — Return true or false according to current state of selector (whole button)

Usage example: button get state and disable/enable 

    var example_selector_1 = new LxSquareSelector({...});

    ...

    if (example_selector_1.disabled()){
        example_selector_1.enable();
    } else {
        example_selector_1.disable();
    }
disableItem(value) value, string Disable item
enableItem(value) value, string Enable item
itemDisabled(value) value, string Return true or false according to current state of passed item

Usage example: item get state and disable/enable 

    var example_selector_2 = new LxSquareSelector({...});

    ...

    if (example_selector_2.itemDisabled('orange')){
        example_selector_2.enableItem('orange');
    } else {
        example_selector_2.disableItem('orange');
    }
rebuild(list, active_value) list, array
of objects;
active_value,
string 		Rebuild list

Usage example: rebuild whole list and set new active value

    var example_selector_3 = new LxSquareSelector({...});
    ...
    var new_list = [
        {
            value: 'owl',
            display: 'Owl',
            line: 'Polar owl'
        },
        {
            value: 'sparrow',
            display: 'Sparrow',
            line: 'Jack the Sparrow'
        },
        {
            value: 'goose',
            display: 'Goose',
            line: 'Gray goose'
        },
        {
            value: 'martlet',
            display: 'Martlet',
            line: 'Fast martlet'
        }
    ];
    example_selector_3.rebuild(new_list, 'sparrow');
